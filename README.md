# 🛒 OneStop Smart Checkout System

## 📌 Overview
This project is a front-end prototype of an optimized e-commerce checkout system designed to reduce cart abandonment. It focuses on improving user experience and implementing cart recovery strategies commonly used in modern online retail platforms.

## 🎯 Objectives
- Reduce cart abandonment rates  
- Improve checkout efficiency and usability  
- Provide a seamless purchasing experience  
- Simulate real-world e-commerce features  

## 🚀 Features
- **One-page checkout** for faster transactions  
- **Guest checkout option** to eliminate sign-up barriers  
- **Dynamic pricing system** with discount logic  
- **Cart abandonment detection** using inactivity tracking  
- **Automated recovery popup** to re-engage users  
- **Real-time UI updates** for pricing transparency  

## 🛠 Technologies Used
- HTML  
- CSS  
- JavaScript  

## ⚙️ How It Works
The system monitors user activity and detects inactivity during the checkout process. If the user becomes inactive, a recovery popup is triggered to encourage them to complete the purchase. Users can also apply a discount code, and the system dynamically updates the total price while ensuring pricing transparency.

## 📊 Purpose
This prototype demonstrates how UX optimization and simple automation techniques can significantly reduce cart abandonment in e-commerce platforms.


## 🧪 Future Improvements
